# PROJETO

### Desenvolvido por aluno do curso Academia do Progamador (2025)

## COMO FUNCIONA?

Nesse clássico jogo da forca, o jogador tem 5 chances para acertar a palavra, que como a dica sugere, é um animal.

## ENTRADA

O jogador interage com o console chutando letras que acredita estarem na palavra. O console, por sua vez, reage incluindo a letra na palavra - quando o chute foi certeiro - ou acrescentando um membro do jogador á forca e acrescentando o dado de erro.
O jogo chega ao fim quando o jogador acerta todas as letras ou erra cinco vezes. O progama é eficiente e permite ao jogador continuar rejogado com outras palavras quantas vezes quiser.

## FUNCIONALIDADES
- Checagem e validação  de entradas.
- Contabilização de erros e condição de vitória.
- Loop que possibilita novas partidas.
- Desenho em tempo real do personagem na forca.
- Banco de palavras secretas.

# COMO USAR

Clone o Repositório
```
git clone https://github.com/xleonardoxmed/JogodaForca.ConsoleApp.git
```
Navegue até a pasta raiz da solução
```
cd jogoDeAdivinhacao
```
Restaure as dependências
```
dotnet restore
```
Navegue até a pasta do projeto
```
cd jogoDeAdivinhacao.ConsoleApp
```
Execute o projeto
```
dotnet run
```
